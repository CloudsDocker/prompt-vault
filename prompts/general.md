# General Prompts

## Explain Code

```
Explain what this code does in plain English. Assume I'm a senior engineer
but unfamiliar with this specific codebase.
```

## Write Tests

```
Write unit tests for this function. Cover happy path, edge cases, and error cases.
Use the same testing framework already in the project.
```

## Debug

```
This code is producing the wrong output. Here is the input, expected output,
and actual output. Identify the root cause and suggest a fix.
```
